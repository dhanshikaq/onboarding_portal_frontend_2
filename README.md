# Quadra Client Onboarding Portal

## Setup Instructions

### Adding the Company Logo

1. **Place the logo file**: Add your `quadrant-logo.png` file to the `public/` folder
2. **File naming**: Ensure the file is named exactly `quadrant-logo.png`
3. **Logo format**: The logo should be a PNG with a transparent or white background

### Running the Application

```bash
cd onboarding_portal_frontend_2
npm install
npm start
```

## Features

- Modern dark theme UI
- Clean, professional design
- Responsive layout
- Quick action menu
- Chat interface
